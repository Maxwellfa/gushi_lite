# 古诗文小助手（微信小程序）

这个是[学古诗](https://xuegushi.cn) 网站的微信小程序。
通过小程序可以更方便简洁的学习中国传统古诗文，希望能对大家有所帮助。

## 样式
使用微信官方的[WeUi](https://weui.io)

## 搜索部分
搜索部分是在[天未](https://github.com/mindawei)的[wsSearchView](https://github.com/mindawei/wsSearchView)的基础上进行的改造和优化。
